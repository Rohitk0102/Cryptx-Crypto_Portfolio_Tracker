# CryptX Web3 Requirements

This document lists all system prerequisites and project dependencies needed to run the CryptX Web3 project on another device.

## 1. System Requirements

Ensure the following software is installed on your machine:

-   **Node.js**: v18.0.0 or higher (Current LTS recommended)
-   **npm**: v9.0.0 or higher (comes with Node.js)
-   **PostgreSQL**: v14 or higher (Database)
-   **Redis**: v6 or higher (Optional, for caching - set DISABLE_REDIS=true in .env to skip)
-   **Git**: For version control

## 2. Global Dependencies (Dev)

These tools are recommended for development:

-   `nodemon` (via `npm install -g nodemon`)
-   `ts-node` (via `npm install -g ts-node`)
-   `prisma` (via `npm install -g prisma`)

## 4. Installation Instructions

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd Cryptx_web3
    ```

2.  **Run Setup Script:**
    This script will install dependencies, create environment files, and set up the database.
    ```bash
    ./start.sh setup
    ```

    *Note: If you are on Windows, use the provided batch file:*
    ```batch
    start.bat setup
    ```

3.  **Manual Setup (Alternative):**
    If the script doesn't work for you:
    ```bash
    # Install dependencies
    npm install

    # Setup Environment
    cp apps/api/.env.example apps/api/.env
    cp apps/web/.env.example apps/web/.env.local
    # (Update .env files with your credentials)

    # Setup Database
    cd apps/api
    npx prisma generate
    npx prisma migrate dev
    ```

4.  **Run Development Server:**
    ```bash
    ./start.sh
    # OR
    npm run dev
    ```
